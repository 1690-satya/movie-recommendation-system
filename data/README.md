# Data Directory

Place the MovieLens dataset files here:

## Required Files

1. **movies.csv** - Movie metadata
   - Columns: `movieId`, `title`, `genres`

2. **ratings.csv** - User ratings
   - Columns: `userId`, `movieId`, `rating`, `timestamp`

## Download

Get the dataset from: https://grouplens.org/datasets/movielens/

Recommended: **MovieLens Latest Small** (ml-latest-small.zip)

## Note

Data files are excluded from git (see .gitignore) due to size. Users must download separately.
